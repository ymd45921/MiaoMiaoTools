## 使用说明

按照以下步骤启动 `venv`，并且安装所需的依赖：

在 Linux/MacOS 上：
```shell
source .venv/bin/activate
pip install -r requirements.txt
```

在 Windows 上：
```shell
.venv\Scripts\activate
pip install -r requirements.txt
```

然后就可以使用 `python -m` 命令运行此目录下的小工具了。
